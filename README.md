#Gara Operator System

## Required
 - Npm
 - bower
 - composer

## Installtion
 - composer install
 - npm install
 - bower install

## Configuration

  - cp .env.example .env
  - vim .env

## Migration & Seediing
 - php artisan migrate:refresh --seed
